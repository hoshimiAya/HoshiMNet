
# 结构

## 类

### Channel

- 网络文件描述符: fd
- 需要监听的事件类型: events
- 监听到的事件类型: revents
- 当前所属的EventLoop: loop
- 事件的回调函数: readCallback, writeCallback, closeCallback, errorCallback

- 回调函数
- 设置4个事件的回调函数
- 获取当前的 fd 和 events
- 设置 events 和 revents

### Epoller

- epoll文件描述符
- 管理多个Channel
- 当前所属的EventLoop

- 获取多个epoll事件的监听结果

### EventLoop

- 启动事件循环

### Acceptor

- 绑定服务器InetAddress
- 管理服务器Channel

- 收到新连接的回调函数
- 监听新连接

### Connector

### Buffer

### Socket

### InetAddress

- ip地址
- 端口号

### TcpServer

- 绑定EventLoop
- 绑定InetAddress
- 管理Acceptor

- 设置Acceptor收到新连接的回调函数
- 启动服务器
